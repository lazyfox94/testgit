Các bước đẩy code lên Github:

  • Nếu lần đầu đẩy code:
    1. Bước 1: Tải & cài đặt git bash
    2. Bước 2: Chọn tùy chọn cho VS code làm default
    3. Bước 3: Tắt toàn bộ VS code đang mở
    4. Bước 4: Mở lại VS code thư mục cần upload code lên github
    5. Bước 5: Mở terminal PowerShell
    6. Bước 6: Kiểm tra xem git bash đã được cài đặt thành công chưa bằng "git --version"
    7. Bước 7: Nếu đã cài đặt thành công thì gõ "git init" để khởi tạo
    8. Bước 8: Khởi tạo reposity trên Github để lưu code, chọn public
    9. Bước 9: Copy đường link remote đến repo với link có dạng "remote add origin http...."
    10. Bước 10: Kiểm tra xem đã add được remote chưa bằng câu lệnh "git remote -v"
    11. Bước 11: Thêm thư mục local vào repo qua câu lệnh "git add ."
    12. Bước 12: Thêm comment mô tả cho thư mục bằng "git commit -m "đây là cmt"
    13. Bước 13: Đẩy toàn bộ thư mục code vào repo bằng "git push origin main"
    14. Bước 14: Refresh trang repo của Github để check kết quả
  • Nếu đẩy code các lần sau:
    1. Bước 1: Nếu có sửa code cũ để đẩy lại thì sửa & save code
    2. Bước 2: Thêm toàn bộ thư mục chứa code vào repo bằng "git add ."
    3. Bước 3: Thêm comment cho những thay đổi hoặc mô tả thư mục chứa code bằng "git commit -m "đây là cmt"
    4. Bước 4: Đẩy toàn bộ thư mục code vào repo bằng "git push origin main"
    5. Bước 5: Refresh trang repo của Github để check kết quả

Vậy là xong hướng dẫn đẩy code từ local vào Github qua VS code!
